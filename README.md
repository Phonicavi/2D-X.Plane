# 2D-X.Plane  

	微信打飞机 Cocos2D-X实现  
	C++  
	只测试了iPhone5s(iOS 9.3)  


## 版本历史  


### V1.3 (2016.07.19)  

#####更新玩法:  

> 这个游戏要从弹幕射击变成塔防了吗？

   * 更新了玩法 目标是守卫后防线  
   * 难度进一步上升 需要在10颗星的限度内坚持完5个level  
   * 每升级一个level后机动性能都有提高  
   * 嫌missile和explosion声音吵的话可以点左上角关闭音效  
   * 欢迎意见和建议
